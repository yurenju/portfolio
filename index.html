Work

Build system of Firefox OS Gaia

Firefox OS is a web based and open-source operating system for smartphones, tablets and smart TVs. Gaia is the user interface of Firefox OS, which includes window manager, notification bar, homescreen and default core apps. Gaia build system is responsible for generating preload data for the device, customization mechanism, localization, optimization and a part of testing mechanism.

Responsibilities: I'm candidate of gaia build system module owner, proposed customization mechanism and refactoring plan for more reliable architecture, also added unit/integration test for build system.

Skills: Javascript, Makefile, xpcshell, Testing (Mocha)

Related Links: Github, Firefox OS Module, Gaia Build System Roadmap
